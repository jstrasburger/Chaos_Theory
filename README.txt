This R file is my attempt to visualize and understand Chaos Theory. It demonstrates some key terms and phrases: sensitive dependence on initial conditions, Feigenbaum's Constant, period doubling bifurcations, a part of the mandelbrot set, and logistics map. This code was 
inspired by Veratasium's popular video (https://www.youtube.com/watch?v=ovJcsL7vyrk&t=21s).
